# Trae 编程实战教程

本目录是一套从环境搭建到综合项目落地的 Trae 编程系统课程，覆盖前端、后端、数据库、AI 集成、微服务、DevOps 与团队协作等典型工程场景，按 5 个部分共 20 章组织，配套对外课程提纲与首章 PPT。完整大纲见 [《Trae 编程实战》课程提纲（对外）.md](《Trae%20编程实战》课程提纲（对外）.md)。

## 课程结构

| 部分                                                          | 章节范围    | 主题             |
| ------------------------------------------------------------- | ----------- | ---------------- |
| [第一部分 - Trae 基础入门](第一部分-Trae基础入门/)            | 第 1–4 章   | 环境配置与入门   |
| [第二部分 - 常见编程场景实战](第二部分-常见编程场景实战/)     | 第 5–8 章   | Web 全栈开发     |
| [第三部分 - 高级应用场景](第三部分-高级应用场景/)             | 第 9–12 章  | AI、实时与微服务 |
| [第四部分 - 团队协作与最佳实践](第四部分-团队协作与最佳实践/) | 第 13–16 章 | 工程化与运维     |
| [第五部分 - 综合项目实战](第五部分-综合项目实战/)             | 第 17–20 章 | 端到端项目交付   |
