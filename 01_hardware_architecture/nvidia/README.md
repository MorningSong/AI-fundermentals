# NVIDIA 硬件架构与算力解析

## 1. 概述

本目录聚焦于 AI 算力基础设施的核心——硬件加速器架构，重点提供 GPU 的深度技术解析以及不同 AI 芯片（如 GPGPU 与 NPU）在大型语言模型训练和推理场景下的选型对比。旨在帮助开发者和系统架构师深入理解底层硬件设计原理，从而在实际工程中做出更优的性能调优与技术决策。

## 2. 核心内容导航

本章节整理了当前目录下的核心文档与子模块，涵盖了从基础硬件特性剖析到宏观算力生态对比的完整脉络。

### 2.1 深入理解 GPU 架构

本小节系统性地介绍了 GPU 的底层硬件特性、内存模型以及不同产品线的演进，内容由 `understand_gpu_architecture` 目录提供，详细请查阅其 [README 文档](understand_gpu_architecture/README.md)。

- **架构基础**：详细解析 GPU 与 CPU 的设计理念差异、并行计算机制以及复杂的内存层次结构（全局内存、共享内存等）。
- **硬件实例分析**：通过剖析 Tesla V100（数据中心级）与 RTX 5000（工作站级）的具体架构，探讨 Tensor Core 与 RT Core 的实际应用。
- **实践练习**：提供可执行的 CUDA API 查询与内存带宽基准测试，帮助开发者动手验证理论知识。

### 2.2 大模型算力架构对比

本小节专注于在大型语言模型时代，面对海量参数与指数级增长的计算需求，不同架构芯片的优劣势分析与选型指导。

- **[GPGPU vs NPU：大模型推理与训练的算力选择指南](GPGPU_vs_NPU_大模型推理训练对比.md)** - 深度对比通用图形处理器与神经网络专用处理器的架构差异
  - **技术架构解析**：深入拆解以 NVIDIA H100 为代表的 GPGPU 架构与以华为昇腾 910B 为代表的 NPU 架构设计。
  - **大模型适配性**：针对 Transformer 架构的注意力机制加速、混合精度训练以及内存管理策略进行专项分析。
  - **场景化选型建议**：结合训练与推理的不同需求，提供在成本效益、生态成熟度与部署复杂度等维度的实用决策指导。
